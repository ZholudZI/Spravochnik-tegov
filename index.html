Справочник тегов

Файл с расширением .html
открываются в браузере

  • Теги для текста
  • Теги для выделения текста
  • Графические теги
  • Теги задающии структуру сайта

1. Теги для текста:

  • p - тег абзаца
  • h1..h6 - теги заголовка
  • ul/ol - нумерованный и маркированный список
  • li - элемент списка
  • br - тег переноса строки
  • a - ссылка, от англ. anchor
  • Подробнее

2. Теги для выделения текста:

  • b - жирный текст
  • i - курсивный текст
  • del - зачёркнутый текст
  • mark - выделенный жёлтым фоном текст
  • sub - индексный текст

3. Графические теги

  • hr - тег горизонтальной линии
  • img - тег изображения

4. Теги задающии структуру сайта

  • !DOCTYPE html - не тег, а инструкция для сайта
  • html - тег заключает в себе всё содержимое сайта
  • head - тег заключает в себе служебную информацию сайта
  • body - тег контента сайта
  • div - уневерсальный блок сайта
  • section - задаёт разделы сайта
  • main - основней контент, стоит после header и включает в себя весь контент
